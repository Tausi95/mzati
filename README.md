A high-converting one-page website proposal for **MZATI**, a pediatric therapy center providing speech therapy, physiotherapy, and developmental support for children.

## What This Is

A production-ready static landing page built to:

- Present a **3-month digital growth pilot** to MZATI
- Demonstrate the visibility gap and growth opportunity
- Convert interest into booked conversations at **$300/month**

## Tech Stack

- Pure HTML + CSS — no frameworks, no build step
- Google Fonts (Inter)
- Intersection Observer API for scroll animations
- Fully responsive (mobile, tablet, desktop)
